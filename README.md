## how to use

2. `git clone`
1. `npm install`
2. `npm build`
3. visit chrome extensions menu (chrome://extensions)
4. check developer mode checkbox
5. load unpacked extension
6. visit a tab
7. select some things
8. click the addon icon (it's the 'hello' default right now)
